Fallback translation

Fallback translation

Locale the following:

var locales = {
  en: {
    message: 'hello world'
  },
  ja: {
  }
}
Vue.config.lang = 'ja'

Template the following:

<p>{{ $t('message') }}</p>

Output the following:

<p>hello world</p>
